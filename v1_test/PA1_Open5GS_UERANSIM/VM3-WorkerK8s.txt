Bước 1 – Cài Kubernetes (VM2 = Master, VM3 = Worker)				
Chúng ta chia bước này thành 4 việc:				
1	Cài containerd trên cả VM2 & VM3			
2	Cài kubeadm, kubelet, kubectl trên cả hai VM			
3	Init Kubernetes trên MasterK8s (VM2)			
4	Join WorkerK8s (VM3) vào cluster			
Mình bắt đầu Việc 1.				
🧩 Việc 1 – Cài containerd (runtime) trên VM2 & VM3				
Kubernetes cần container runtime. Chúng ta dùng containerd vì ổn định và tương thích tốt với K8s.				
🔧 1. Cài containerd				
📌 Chạy lệnh này trên VM2 (MasterK8s) và VM3 (WorkerK8s):				
apt update				
apt install -y containerd				
🔧 2. Tạo file cấu hình mặc định cho containerd				
mkdir -p /etc/containerd				
containerd config default | tee /etc/containerd/config.toml				
🔧 3. Bật SystemdCgroup để tương thích Kubernetes				
Mở file:				
nano /etc/containerd/config.toml				
Tìm đoạn:				
[plugins."io.containerd.grpc.v1.cri".containerd.runtimes.runc.options]				
  SystemdCgroup = false				
→ Sửa thành:				
  SystemdCgroup = true				
Lưu lại (Ctrl+O, Enter, Ctrl+X).				
🔧 4. Restart containerd				
systemctl restart containerd				
systemctl enable containerd				
🔧 5. Cài module kernel cần thiết cho Kubernetes				
cat <<EOF | tee /etc/modules-load.d/k8s.conf				
overlay				
br_netfilter				
EOF				
Nạp module:				
modprobe overlay				
modprobe br_netfilter				
🔧 6. Bật các sysctl cho K8s				
cat <<EOF | tee /etc/sysctl.d/k8s.conf				
net.bridge.bridge-nf-call-iptables = 1				
net.bridge.bridge-nf-call-ip6tables = 1				
net.ipv4.ip_forward = 1				
EOF				
Áp dụng:				
sysctl --system

🧩 VIỆC 2 – Cài kubeadm, kubelet, kubectl (trên cả Master & Worker)				
🔧 Bước 1: Cài dependency				
apt update				
apt install -y apt-transport-https ca-certificates curl				
🔧 Bước 2: Thêm kho Kubernetes				
curl -fsSL https://pkgs.k8s.io/core:/stable:/v1.30/deb/Release.key | gpg --dearmor -o /etc/apt/keyrings/kubernetes-apt-keyring.gpg				
echo "deb [signed-by=/etc/apt/keyrings/kubernetes-apt-keyring.gpg] https://pkgs.k8s.io/core:/stable:/v1.30/deb/ /" | tee /etc/apt/sources.list.d/kubernetes.list				
(Theo đúng tài liệu chính thức Kubernetes)				
🔧 Bước 3: Cài kubeadm, kubelet, kubectl				
apt update				
apt install -y kubelet kubeadm kubectl				
🔧 Bước 4: Hold version (tránh auto-update gây lỗi)				
apt-mark hold kubelet kubeadm kubectl				
🔧 Bước 5: Kiểm tra cài đặt				
kubeadm version				
kubectl version --client				
kubelet --version				
Nếu hiện phiên bản tương ứng → OK.
